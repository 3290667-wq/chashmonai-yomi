## סקירה כללית

**חשמונאי יומי** היא אפליקציית לימוד תורני מותאמת לחיילי חטיבת חשמונאים.

### מטרות האפליקציה:
- עידוד לימוד יומי קבוע
- מעקב אחר התקדמות אישית
- יצירת קהילת לומדים
- תגמול על התמדה בלימוד

### טכנולוגיות:
- **Frontend:** Next.js 16, React, TailwindCSS
- **Backend:** Next.js API Routes
- **Database:** PostgreSQL (Neon)
- **Authentication:** NextAuth v5
- **Hosting:** Vercel
- **PWA:** התקנה כאפליקציה על הטלפון

## ממשק ניהול

### גישה למנהלים בלבד

**נתיב ראשי:** `/admin`

### 1. ניהול משתמשים (`/admin/users`)
| פעולה | תיאור |
|-------|-------|
| צפייה בכל המשתמשים | רשימה מלאה עם חיפוש |
| שינוי תפקיד | USER / RAM / ADMIN |
| שיוך לפלוגה | הגדרת פלוגה למשתמש |
| מחיקת משתמש | הסרה מהמערכת |
| איפוס סיסמה | שליחת סיסמה חדשה |

### 2. ניהול תוכן (`/admin/content`)
| פעולה | תיאור |
|-------|-------|
| הוספת תוכן | יצירת תוכן חדש |
| עריכת תוכן | שינוי תוכן קיים |
| מחיקת תוכן | הסרת תוכן |
| סוגי תוכן | סרטון / חסידות / מוסר / מחשבה / מאמר |

### 3. ניהול בקשות מימוש (`/admin/assignments`)
| פעולה | תיאור |
|-------|-------|
| צפייה בבקשות | כל בקשות המימוש |
| אישור בקשה | אישור מימוש נקודות |
| דחיית בקשה | דחייה עם הערה |

### 4. הגדרות מערכת (`/admin/settings`)
| הגדרה | תיאור |
|--------|-------|
| נקודות לדקה | כמה נקודות לכל דקת לימוד |
| מינימום דקות | מינימום דקות לצבירת נקודות |
| בונוס רצף | נקודות על שמירת רצף |
| בונוס השלמה | נקודות על סיום לימוד |
| שם האפליקציה | שינוי שם האפליקציה |
| סלוגן | שינוי הסלוגן |

## תפקידים והרשאות

### USER (משתמש רגיל)
- גישה ללימוד יומי
- צפייה בזמנים
- צבירת ומימוש נקודות
- צ'אט עם רבנים
- עריכת פרופיל

### RAM (רב/ראם)
כל הרשאות USER, ובנוסף:
- גישה לניהול תוכן (לפלוגה שלו)
- צפייה בהודעות מהחיילים
- הוספת תוכן לפלוגה

### ADMIN (מנהל ראשי)
כל ההרשאות, כולל:
- ניהול כל המשתמשים
- ניהול כל התוכן
- אישור בקשות מימוש
- הגדרות מערכת
- גישה מלאה לכל הפלוגות

## אפשרויות להרחבה

### פיצ'רים מומלצים להוספה:

#### 1. התראות Push
```
- התראה על זמן תפילה
- תזכורת ללימוד יומי
- הודעה על תוכן חדש
- עדכון על נקודות שנצברו
```

#### 2. לוח תוצאות (Leaderboard)
```
- דירוג לפי נקודות
- דירוג לפי רצף
- תחרות בין פלוגות
- פרסים שבועיים/חודשיים
```

#### 3. מבחנים ובחנים
```
- שאלות על החומר הנלמד
- בונוס נקודות על ציון גבוה
- מעקב אחר הבנת החומר
```

#### 4. לוח שנה יהודי
```
- תאריכים עבריים
- פרשת השבוע
- חגים ומועדים
- ספירת העומר
```

#### 5. הקלטות ופודקאסטים
```
- שיעורי אודיו
- הרצאות מוקלטות
- פודקאסט יומי
```

#### 6. קבוצות לימוד
```
- חברותות וירטואליות
- לימוד בזוגות
- פורום שאלות ותשובות
```

#### 7. משימות יומיות
```
- משימות מותאמות אישית
- אתגרים שבועיים
- מטרות לטווח ארוך
```

#### 8. אינטגרציות
```
- סנכרון עם Google Calendar
- שיתוף ברשתות חברתיות
- ייצוא סטטיסטיקות
```

#### 9. מערכת הישגים (Badges)
```
- תגים על הישגים
- רמות משתמש
- אוספי הישגים
```

#### 10. מצב לילה
```
- תמיכה בדארק מוד
- מעבר אוטומטי לפי שעה
```

## פרטים טכניים

### דרישות מערכת:
- דפדפן מודרני (Chrome, Safari, Firefox, Edge)
- חיבור אינטרנט
- מסך בגודל 320px ומעלה

### PWA (אפליקציה מותקנת):
האתר תומך בהתקנה כאפליקציה:
1. פתח את האתר ב-Chrome
2. לחץ על "הוסף למסך הבית"
3. האפליקציה תופיע כאייקון

### תמיכה במובייל:
- עיצוב מותאם לכל גודל מסך
- תפריט נייד נוח
- כפתורים בגודל מתאים למגע
